# Sonatrio | AI-Powered Content from Your Audio & Video

Sonatrio transforms your audio and video files into high-quality content, including blog posts, social media captions, summaries, and more—powered by AI!

## 🚀 Features
- Upload audio or video files
- AI-driven transcription with **Deepgram**
- Automatic content generation with **GPT**
- Create blog posts, social media snippets, and summaries

## 🛠️ Getting Started
### Prerequisites
Ensure you have the following installed:
- **Node.js** (latest LTS version recommended)
- **npm** or **yarn**
- **Supabase** and **Express** configured

### Installation & Setup
1. **Clone the repository**
   ```sh
   git clone https://github.com/your-repo/sonatrio.git
   cd sonatrio
   ```

2. **Start the Frontend**
   ```sh
   cd frontend
   npm install
   npm run dev
   ```

3. **Start the Backend**
   ```sh
   cd backend
   npm install
   npm run dev
   ```

## ⚡ Usage
1. Upload an audio or video file.
2. Let Sonatrio transcribe and generate AI-powered content.
3. Download or copy your content for blogs, social media, or marketing!
